# SUPABASE SETUP GUIDE

To make ARTiE PRINTz fully functional, follow these steps in your Supabase Dashboard.

## 1. Create the Orders Table

Run this SQL query in the **SQL Editor**:

```sql
-- Orders table (single table with JSONB products column)
CREATE TABLE IF NOT EXISTS orders (
  id          bigint GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  created_at  timestamptz DEFAULT now() NOT NULL,
  customer_name text NOT NULL,
  phone       text NOT NULL,
  whatsapp    text,
  address     text NOT NULL,
  total       numeric NOT NULL,
  status      text DEFAULT 'PENDING',
  notes       text,
  products    jsonb DEFAULT '[]'::jsonb
);

-- Enable Row Level Security (optional but recommended)
ALTER TABLE orders ENABLE ROW LEVEL SECURITY;

-- Allow anonymous inserts (for placing orders without login)
CREATE POLICY "Allow anon insert" ON orders
  FOR INSERT TO anon WITH CHECK (true);

-- Allow anonymous reads (for admin panel)
CREATE POLICY "Allow anon select" ON orders
  FOR SELECT TO anon USING (true);

-- Allow anonymous updates (for status changes)
CREATE POLICY "Allow anon update" ON orders
  FOR UPDATE TO anon USING (true) WITH CHECK (true);
```

## 2. Storage Setup (for print file uploads)

1. Go to **Storage** in Supabase Dashboard.
2. Create a NEW bucket named `prints`.
3. Set the bucket to **Public**.
4. Add an RLS policy to allow public uploads:
   - Policy name: `Allow public upload`
   - Allowed operations: `INSERT`
   - Target roles: `anon`

## 3. Environment Variables

Copy your **Project URL** and **Anon Key** from **Project Settings > API** and set them in your `.env` file:

```env
VITE_SUPABASE_URL="https://your-project.supabase.co"
VITE_SUPABASE_ANON_KEY="your-anon-key"
DISCORD_WEBHOOK_URL="your-discord-webhook-url"
VITE_ADMIN_PASSWORD="your-admin-password"
```

> **For Vercel:** Add these same variables in your Vercel project's **Settings > Environment Variables**.
> The `DISCORD_WEBHOOK_URL` must be added as a server-side env var (no `VITE_` prefix) so it's only accessible by the serverless function.

## 4. Discord Webhook

1. Open Discord.
2. Channel Settings > Integrations > Webhooks.
3. Create a New Webhook and copy its URL.
4. Add it as `DISCORD_WEBHOOK_URL` in both your `.env` file and Vercel env vars.
